
# 🏆 Manual del Héroe: ¡Publica tu Entrenador Saber 11 Gratis!

¡Hola, Campeón! Como Firebase App Hosting pide pagar, vamos a usar **Vercel**. Es el sitio más famoso para publicar estas apps y es **totalmente gratis** para estudiantes como tú.

## 🎒 Paso 1: Tu Mochila Mágica (GitHub)
1. Entra a [github.com](https://github.com) y crea una cuenta.
2. Crea un **"New Repository"** llamado `entrenador-saber11`.
3. Sube tu código ahí (puedes arrastrar las carpetas si usas GitHub Desktop o la web).

## 🏰 Paso 2: Tu Castillo en Internet (Vercel - ¡Gratis!)
Vercel es genial porque no te pide tarjeta de crédito para proyectos personales:
1. Entra a [vercel.com](https://vercel.com) y regístrate usando tu cuenta de **GitHub**.
2. Dale al botón **"Add New"** y luego a **"Project"**.
3. Verás tu mochila de GitHub (`entrenador-saber11`). Dale al botón **"Import"**.
4. **¡IMPORTANTE!**: Vercel te preguntará por "Environment Variables". Ahí debes pegar tu clave de Google (GEMINI_API_KEY) para que la IA funcione.
5. Dale a **"Deploy"**. ¡En 2 minutos tu página estará viva!

## 🚀 Paso 3: ¡Tu URL de Victoria!
Cuando termine, Vercel te dará un enlace como `tu-app.vercel.app`. **¡Esa es tu dirección oficial para compartir con tus amigos!**

---
### 💡 Recordatorios de Misión:
- **Firebase sigue siendo gratis:** No borres tu proyecto de Firebase. La base de datos y el login seguirán funcionando gratis en el plan "Spark". Solo no uses la opción "App Hosting" dentro de Firebase.
- **Modo Admin:** Entra a tu app, dale a tu foto y busca "Panel de Administración".
- **Áreas:** Ya tienes Matemáticas, Lectura Crítica, Sociales, Naturales, Inglés y Socioemocionales listas.
